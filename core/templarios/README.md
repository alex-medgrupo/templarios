# Templarios

## Desenvolvimento

```bash
# npm
cd core
npm run tp:start

# yarn
cd core
yarn tp:start
```

## Produção

```bash
# npm
cd core
npm run tp:build

# yarn
cd core
yarn tp:build
```

### Storybook

```bash
# npm
cd core
npm run tp:build:storybook

# yarn
cd core
yarn tp:build:storybook
```
